# Καλώς ήρθατε στο LogistikoCRM!

## Ευχαριστούμε για το ενδιαφέρον σας να συνεισφέρετε στο project!

### Τρόποι Συνεισφοράς

Η συνεισφορά κώδικα δεν είναι ο μόνος τρόπος να βοηθήσετε. Υπάρχουν πολλές ευκαιρίες:

- Αναφέρετε bugs ή προτείνετε νέες λειτουργίες μέσω GitHub Issues
- Βελτιώστε την τεκμηρίωση
- Βοηθήστε με μεταφράσεις
- Δοκιμάστε νέες εκδόσεις και δώστε feedback

### Ανάπτυξη Κώδικα

Το LogistikoCRM βασίζεται στο Django framework. Αν είστε εξοικειωμένοι με Django, θα βρείτε τον κώδικα κατανοητό.

#### Πριν υποβάλετε Pull Request

1. **Τρέξτε τα tests:**
   ```bash
   python manage.py test
   ```

2. **Ελέγξτε τον κώδικα:**
   ```bash
   python manage.py check
   ```

3. **Βεβαιωθείτε ότι ακολουθείτε το PEP 8**

#### Οδηγίες για Commits

- Γράψτε περιγραφικά commit messages
- Κάντε μικρά, εστιασμένα commits
- Αναφέρετε το σχετικό issue αν υπάρχει (π.χ. "Fixes #123")

### Αναφορά Προβλημάτων

Όταν αναφέρετε ένα bug, παρακαλούμε συμπεριλάβετε:

- Έκδοση Python και Django
- Βήματα αναπαραγωγής του προβλήματος
- Αναμενόμενη vs πραγματική συμπεριφορά
- Screenshots αν είναι χρήσιμα

### Ασφάλεια

Για θέματα ασφαλείας, **μην** ανοίγετε δημόσιο issue. Δείτε το [SECURITY.md](.github/SECURITY.md).

Ευχαριστούμε για τη συνεισφορά σας!
